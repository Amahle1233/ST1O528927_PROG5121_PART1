## Destription:
Chartapp is a java based application made to handle a users registration and login.
The project is focused on ensuring that the user inserts valid information,that meet specific conditions.
This project part of my part 1 for my programming PoE.

## Features
### Username Validation
Ensures that the user only inserts username that has (_) and is not longer than five characters.
### Password Complexity
Validates that the user password is atleast eight characters long,and contains a capital letter,a number and a special character.
### Phone Number Verification
verifies that the cellphone number includes the South African international code (+27) and also that the number is the correct length.
### Automated Testing
Used Junit 5 to test the validation,ensure that the method work as expected
